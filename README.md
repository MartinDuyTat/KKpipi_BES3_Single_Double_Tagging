# KKpipi_BES3_Single_Double_Tagging
This repository contains the BOSS 6.6.4p02 packages for a D->KKpipi analysis and measurement of strong phases

I have also updated it to work with BOSS 7.1.0

### List of tag modes:
* KKpipi
* Kpi
* Kpipi0
* Kpipipi
* KeNu
* KK
* pipi
* pipipi0
* KSpi0pi0
* KLpi0
* KLomega(pipipi0)
* KLeta
* KSpi0
* KSeta
* KSetaPrime(pipieta)
* KSetaPrime(rhogamma)
* KLpi0pi0
* KSpipipi0
* KSpipi
* KLpipi
* KSKK
* KLKK
